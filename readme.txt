== Description ==

Personal Chat Room adds a facility to allow your customers to chat with one another. Here They can questions with another users and can get the answers
from another users of your website.

For any bug or report or error or suggetions or feedback, you can post the support

Basically It has the following features : 
 * Add a sidebar button to your site to allow your customers to chat
 * Add shortcode PERSONAL_CHAT_ROOM to any page or post to activate the plugin
 * Private Chat (Premium Feature)


== Installation ==

Here is what you need to do: 

1. Install plugin OR unzip the downloaded file to /wp-content/plugins/
1. Go to your plugin settings and enable the plugin.
3. Go to "Chat Room" link and save the configuration
3. save your database configuration using dashboard
4. Add `PERSONAL_CHAT_ROOM` shortcode to your header, post, page or widget
